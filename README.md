# Sales ETL Pipeline

## Overview

This project implements an end-to-end ETL (Extract, Transform, Load) pipeline for processing sales data and preparing analytics-ready datasets for reporting and business intelligence.

## Features

* Extract data from CSV source files
* Clean and validate data
* Transform and enrich sales records
* Create analytics-ready datasets
* Load data into SQLite database
* Perform validation and reconciliation checks
* Generate data warehouse schema using SQL
